## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

You need to have Python 3 installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

### Installation

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/S-Leishem/flappy-modi.git
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd flappy-modi
    ```

3.  **Install the required packages:**

    It's recommended to use a virtual environment to keep dependencies isolated.
    ```sh
    # Create and activate a virtual environment (optional)
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate

    # Install dependencies
    pip install pygame
    ```

### Running the Game

Once the dependencies are installed, you can run the game with the following command:

```sh
python main.py
```
*(Note: If your main game file is not `main.py`, replace it with the correct file name.)*

## How to Play

-   Press the **Spacebar** or **click the left mouse button** to make the character jump.
-   Navigate through the gaps in the pipes to score points.
-   The game is over if you collide with a pipe or the ground.
